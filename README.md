# Ende

Ende is a Distance and Durations matrix generator focused on being fast, accurate, and extensible.  

## Installation

```bash
git clone https://github.com/andrewgy8/ende.git
```

## Usage

```bash
$ cargo build
$ ./target/debug/ende
```
